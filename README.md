# Memory Card Game

## Description
This project is a simple Memory Card Game where the player flips cards and tries to find matching pairs. The goal is to match all pairs with the fewest moves and in the shortest time.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open index.html in your browser

## Features
- Card matching system
- Move counter
- Timer
- Restart button
- Win message

## Game Logic
The game uses an array of card pairs which are shuffled randomly.
The player flips two cards:
- If they match → they stay open
- If not → they flip back

## Algorithms
- Shuffle algorithm using random sorting
- Comparison logic for matching cards
